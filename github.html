STEP 1

using git init command or push code

I suggest you to use VS code editor because its a one of the best platform for every type of file


1- First create a folder using mkdir command in VS code.
2- Then create a file any type you want like txt, html, py, go and many more.
3- Now write your code or file whatever you want.
4- Now run command git init.

Open Github and create repo.

5- Now run command git remote add origin (github-repo-link).
6- Then run command git add .
7- Then run command git commit -m "anything"
8- Then run command git push origin main

STEP 2

by clone the repo

1- Go to github and copy the link of the repo which you want to clone.
2- First open Vs code and run command git clone (repo-link).
3- Now write your code or edit file whatever you want.
4- You can also create a file any type you want like txt, html, py, go and many more.
5- You can also create new branch or change
6- Then run command git add .
7- Then run command git commit -m "anything"
8- Then run command git push origin main
