STEP 1

using git init command or push code

I suggest you to use VS code editor because its a one of the best platform for every type of file


  • First create a folder using mkdir command in VS code.

  • Then create a file any type you want like txt, html, py, go and many more.

  • Now write your code or file whatever you want.

  • run command git init
    • .

    Open Github and create repo.

  • Now run command git remote add origin (github-repo-link).

  • Then run command git add .

  • Then run command git commit -m "anything"

  • Then run command git push origin main

STEP 2

by clone the repo

  • Go to github and copy the link of the repo which you want to clone.

  • First open Vs code and run command git clone (repo-link).

  • Now write your code or edit file whatever you want.

  • You can also create a file any type you want like txt, html, py, go and many more.

  • You can also create new branch or change

  • Then run command git add .

  • Then run command git commit -m "anything"

  • Then run command git push origin main
